Tarea 1: Ejercicio básico

Ejercicio Tema 1

Crear un "pen" en la plataforma CodePen que muestre una sencilla página HTML (texto, enlaces, encabezados...) a la que se le dará estilos CSS muy básicos (colores, por ejemplo).

Nota: Recuerda que en la plataforma CodePen el código HTML que se incluye es sólo aquel que está dentro de las etiquetas <body></body>. No es obligatorio incluir el resto de código.
